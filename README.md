
# WELL CONNECT & Freedom CONNECT Firmware

This repository contains firmware and update tools for the **WELL CONNECT** and **Freedom CONNECT** wireless DMX boxes.

---

## ⚠️ Important Information

- **Version Matching:** Always use the box firmware version that matches your phone app version.
- **Hardware Compatibility:** Firmware files are labeled `WELL_CONNECT`, but they are fully compatible with both **WELL CONNECT** and **Freedom CONNECT** hardware.
- **WebUI Access:** The WebUI is typically available only for the first **30 minutes** after the box is powered on.

---

## 🚀 WebUI (Management Interface)
The WebUI is required to upload firmware updates to your box.
